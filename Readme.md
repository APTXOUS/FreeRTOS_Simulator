# DEMO Simlulator Project of FreeRTOS 

This is a demo project of simlulator for FreeRTOS v10.3.1.

The new verison of FreeRTOS changed ```vTaskIncrementTick``` to ```xTaskIncrementTick```.



# How to use

Compile
```shell
make all
```
Run
```shell
./simulator_linux.bin
```
